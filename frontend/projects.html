Featured Project

AWS Cloud Resume Platform

Current Build

A production-style serverless portfolio platform on AWS

Phase 1

The AWS Cloud Resume Platform is a public-facing portfolio site built to demonstrate hands-on experience with core AWS services used in modern cloud environments. It combines static hosting, content delivery, DNS, TLS, a serverless API, database-backed features, IAM permissions, and monitoring.

Phase 1 Scope
  • Static frontend hosted in Amazon S3
  • Global delivery with CloudFront
  • Custom domain with Route 53
  • HTTPS certificate through ACM
  • Visitor counter API with API Gateway and Lambda
  • DynamoDB-backed visitor count storage
  • IAM role permissions and CloudWatch logging
What This Demonstrates
  • AWS fundamentals applied in a real project
  • Serverless architecture concepts
  • Frontend to backend cloud integration
  • Operational awareness through monitoring
  • Version-controlled project development in GitHub

Project Strategy

Why this portfolio is being built in phases

Build Something Real First

The first priority is creating a public, working AWS project that can be shown to recruiters and hiring managers. That means finishing a polished, deployed platform before layering in more advanced tooling.

Add Industry-Recognized Skills in Layers

After the core platform is complete, the same project becomes the foundation for Terraform, CI/CD, security, and Kubernetes work. This creates a much stronger story than isolated practice labs.
